Auth0 Angular + .net Core API

Sample Implementation of an Angular 5 SPA applications that authenticates using Auth0 as an identity provider and securely communicates with a server side application built on .net Core Web API backend.

  • Auth0 official documentation
  • Official Samples. This code is based on the samples provided by Auth0
  • Angular and TypeScript for client-side code
  • ASP.NET Core Web APIs for server-side data and processing

Instructions

  • To execute, start the Client and Server application at the same time
  • Login and create a new registered user
  • Click on the navigation options to verify that the SPA is retrieving protected information from the server
